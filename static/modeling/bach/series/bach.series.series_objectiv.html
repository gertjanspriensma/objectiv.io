Returns the cumulative distribution, that is

(number of partition rows preceding or peers with current row) / (total partition rows).

The value thus ranges from 1/N to 1.

Returns the rank of the current row, without gaps; this function effectively counts peer groups.

window_first_value(window: Union[Window, DataFrame])[source]¶

Returns value evaluated at the row that is the first row of the window frame.

window_lag(window: Union[Window, DataFrame], offset: int = 1, default: Any = None)[source]¶

Returns value evaluated at the row that is offset rows before the current row within the partition; if there is no such row, instead returns default (which must be of the same type as value).

Both offset and default are evaluated with respect to the current row. If omitted, offset defaults to 1 and default to None

window_last_value(window: Union[Window, DataFrame])[source]¶

Returns value evaluated at the row that is the last row of the window frame.

window_lead(window: Union[Window, DataFrame], offset: int = 1, default: Any = None)[source]¶

Returns value evaluated at the row that is offset rows after the current row within the partition; if there is no such row, instead returns default (which must be of the same type as value). Both offset and default are evaluated with respect to the current row. If omitted, offset defaults to 1 and default to None.

window_nth_value(window: Union[Window, DataFrame], n: int)[source]¶

Returns value evaluated at the row that is the n’th row of the window frame (counting from 1); returns NULL if there is no such row.

window_ntile(window: Union[Window, DataFrame], num_buckets: int = 1)[source]¶

Returns an integer ranging from 1 to the argument value, dividing the partition as equally as possible.

window_percent_rank(window: Union[Window, DataFrame])[source]¶
Returns the relative rank of the current row, that is

(rank - 1) / (total partition rows - 1).

The value thus ranges from 0 to 1 inclusive.

window_rank(window: Union[Window, DataFrame])[source]¶

Returns the rank of the current row, with gaps; that is, the row_number of the first row in its peer group.

window_row_number(window: Union[Window, DataFrame])[source]¶

Returns the number of the current row within its partition, counting from 1.
