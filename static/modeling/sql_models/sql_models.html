sql_models.model

Models: * SqlModel - Actual sql-model. Instances of this class form the nodes in the model graph * SqlModelSpec - Specification class, that specifies basic properties of an SqlModel instance * SqlModelBuilder - Helper class to instantiate the (immutable) SqlModel objects. Generally models should extend this class and not the SqlModel class. * CustomSqlModel - Utility child of SqlModelSpec that can be used to add a node with custom sql to a model graph.
