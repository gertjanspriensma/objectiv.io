buhtuh.merge.merge(left: buhtuh.pandasql.BuhTuhDataFrame, right: Union[buhtuh.pandasql.BuhTuhDataFrame, buhtuh.series.series.BuhTuhSeries], how: str, on: Optional[Union[str, List[str]]], left_on: Optional[Union[str, List[str]]], right_on: Optional[Union[str, List[str]]], left_index: bool, right_index: bool, suffixes: Tuple[str, str]) → buhtuh.pandasql.BuhTuhDataFrame[source]¶

Join the left and right Dataframes, or a DataFrame (left) and a Series (right). This will return a new DataFrame that contains the combined columns of both dataframes, and the rows that result from joining on the specified columns. The columns that are joined on can consists (partially or fully) out of index columns.

If the column names on the left and right conflict, then the suffixes are used to distinguish them in the resulting DataFrame. The algorithm for determining the resulting columns and their names is similar to Pandas, but has slight differences when joining on indices and column names conflict.

Parameters

  • left – left DataFrame

  • right – DataFrame or Series to join on left

  • how – supported values: {‘left’, ‘right’, ‘outer’, ‘inner’, ‘cross’}

  • on – optional, column(s) to join left and right on.

  • left_on – optional, column(s) from the left df to join on

  • right_on – optional, column(s) from the right df/series to join on

  • left_index – If true uses the index of the left df as columns to join on

  • right_index – If true uses the index of the right df/series as columns to join on

  • suffixes – Tuple of two strings. Will be used to suffix duplicate column names. Must make column names unique

Returns

A new Dataframe. The original frames are not modified.
